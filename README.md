# DevTerm Desktop

Desktop GUI application for developer utilities.

## Installation

```bash
pip install devterm-desktop
```

## Usage

```bash
# Start the desktop app
devterm-desktop
```

## Features

- **Modern Dark UI** - Built with CustomTkinter
- **8 Tools** - JSON, Base64, URL, Hash, UUID, Password, QR, Case
- **Sidebar Navigation** - Easy tool switching
- **Desktop Native** - Runs as a standalone application

## Tools

- JSON Formatter
- Base64 Encoder/Decoder
- URL Encoder/Decoder
- Hash Generator (MD5, SHA-256, SHA-512)
- UUID Generator
- Password Generator
- QR Code Generator
- Case Converter
